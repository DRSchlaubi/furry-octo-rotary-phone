inline fun <E : Event> Conversation.on(scope: CoroutineScope = this, noinline consumer: suspend E.() -> Unit): Job
Content copied to clipboard

Convenience method that will invoke the consumer on every event E created by Conversation.events.

The events are buffered in an unlimitedbuffer and launched in the supplied scope, which is Conversation by default. Each event will be launched inside the scope separately and any thrown Throwable will be caught and logged.

The returned Job is a reference to the created coroutine, call Job.cancel to cancel the processing of any further events.
